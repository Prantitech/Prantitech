O que é o HTML?


Vamos primeiro entender o significado por trás de seu nome, HTML significa HiperText Markup Language, que seria nada maisque Linguagem de Marcação de Hipertexto em português, seguindo adiante então é possivel compreender que o HTML é o componentebásico para criar sites, sendo uma linguagem simples, direta e funcional para iniciantes e para aqueles que já estãoprogramando a mais tempo, uma linguagem necessária e básica para qualquer programador.

O html consiste em uma série de elementos, estes que serão necessário para organizar nosso código, nosso site. Os elementos irão separar partes do nosso código, "está parte é um paragráfo", "essa linha é um título", "Esse pedaço de código é umatabela", entre outros.


Elementos do HTML


Um elemento de HTML só existirá caso exista um início de tag, um conteúdo e o final da tag. De forma bem simples podemos demonstrar um elemento básica.

< h2 > Site < /h2 >

O h2 dentro dos sinais "maior que" e "menor que" terá a função de dizer para o nosso código que a palavra a seguir é umtítulo, portanto tudo oque existir a partir do "< h2 >" terá uma fonte maior em relação as outras e será colocada no cantoesquerdo do mesmo, já o < /h2 > informará para o nosso código que o título acabou, fazendo o código ir para próximos elementosque contém funções diferentes. E isso por si só é um elemento do HTML.


Tags HTML


As tags do HTML são a base dessa linguagem, elas tem a função de formar um elemento e apontar para o nosso código aonde cada palavra se encaixa, apontar para o nosso navegador como o nosso site sse organiza, elas tem uma nomeclatura padrão:

< body > < /body >

É uma tag, não um elemento! Isso se deve ao fato de não ter um contéudo no meio dessas tags. Podemos perceber que elas estão dentro de 2 sinais ("menos que", "mais que"), e isso servirá de lei para todas as tags, todas elas estarão dentro desses sinais.

Dentro do HTML existem 2 tipos diferentes de tags, com fechamento e sem fechamento, tags com fechamento serão tags deconteúdo, ou seja, irão mostrar algo na tela na forma como queremos, e as tags sem fechamento terão a função de indicar algumdado para o nosso site, como o link para uma imagem ou fonte.


Estrutura do nosso código


Head


Para elaborar o nosso código é necessário 2 tags básicas, o que molda o corpo do nosso site, e então existirá a cabeça donosso código, assim como o nome da tag, head. Podemos pensar na nossa cabeça como uma parte do corpo que manda ações, reações,pelo nosso corpo, é o comando, o que organiza tudo, e essa será a função da tag head, importar imagens, fontes, outros arquivosque terão relação com o nosso site para melhorar o visual de nosso site, expandir mais funcionalidades e ter um conteúdo maisdiferenciado, não só sendo texto e mais texto.

< head >

Conteúdo

< /head >

Dentro dessa tag head são utilizados algumas tags específicas, mas podemos citar 3 principais:

< title > < /title >, essa tag terá a função de impôr um título para o nosso site.

< meta > < /meta >, essa tag irá definir informmações de metadados.

< link > < /link >, especifica as relações entre o documento atual e um recurso externo.


Body


O body vai ser o corpo do nosso site, (assim como o nome diz), uma tag que vai englobar tudo o que será mostrado no nossopágina da internet, percebe-se então que o "head" só passará informação para a nossa página, praticamente quase nada queaparecerá no nosso site, tudo debaixo dos panos, já o body será realmente o conteúdo por si só.

< body >

Conteúdo

< /body >
