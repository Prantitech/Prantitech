Os importantes atributos HTML

Existem na linguagem html atributos globais, estes que podem ser utilizados nos elementos htmls. Eles irão ter a função de editar, nomear, personalizar, esconder ou guardar específicas partes do nosso site, trazendo mais diferenciação e interação no site. Todos os atributos vão ser inseridos dentro dos elementos, sendo assim você terá que especificar a quais irá atribuir, é importante afirmar que alguns desses atributos já estão definidos por padrão mesmo que você não cite eles. Os atributos podem ser booleanos (true ou false), podem ser enumerados ou até terem informações ditas pelo programador.


Como atribuir os elementos:


<p class="exemplo"> texto genérico </p>

<a href="https://prantitech.vercel.app/"> Clique aqui para ser feliz </a>

<p translate="no"> Não traduza isso! </p>

Percebe-se então que o atributo é o "contenteditable" este que é colocado como true (antes sendo falso por padrão), percebe-se também que ele esta situado dentro do elemento < p > de conteúdo "ely dosse todu orredu". Então confirmamos que os atributos sempre seguirão esta sintaxe, sempre sendo colocados dentro dos elementos, e especificamente dentro da primeira parte do elemento (aqueles sem o /).

Atributo Descrição
accesskey O atributo accesskey fornece uma dica para gerar um atalho de teclado no elemento que você utilizar. É necessário que o botão seja um caracter imprimível, e dependendo do navegador irá ser diferente a forma de acessar o botão atribuido.
class O atributo class vai criar uma classe para referenciar um ou varios elementos dentro de um arquivo HTML, permitindo acessar elementos especificos, o JavaScript também percebe o atributo class.
contenteditable O atributo contenteditable é um atributo que indica se o elemento marcado com ele pode ser editado pelo usuário, e se for verdadeiro o navegador irá editar para tal.
id O atributo id cria um ID para especificos elementos para facilitar a diferenciação e leitura do arquivo.
title O atributo title, vai fornecer informações, orientações, relacionados ao elemento em que foi colocado.
style O atributo contém informações de estilo para um documento, servindo assim como um atributo para personalizar seu site. Geralmente referencia documentos CSS.
translate O atributo translate irá especificar se o elemento atribuido deve ser traduzido ou não, na hora de visualizar o site.
tabindex O atributo tabindex, irá organizar a ordem de elementos em que a página irá preferenciar ao apertar a tecla tab.
lang O atributo lang irá especificar a linguagem do elemento marcado.
hidden O atributo hidden é um atributo que pode ser ou falso ou verdadeiro, e que irá servir para esconder ou não partes do site, sejam eles não relevantes ou até mesmo "desbloqueáveis", por exemplo, aparecer outras coisas após fazer um login no site.
draggable O atributo draggable afirma se o elemento pode ou não ser arrastado.
dir O atributo dir especificará a direção do texto em um elemento.
spellcheck O atributo spellcheck terá a função de informar aonde ocorre um erro ortográfico, utilizando as linhas onduladas vermelhas embaixo do texto, é importante citar o fato de que ele só funcionará caso seja um texto feito pelo usuário, portanto ele não ira citar os erros ortográficos feitos dentro do elemento do arquivo do site.
data O atributo data terá a função de guardar dados privados personalizados do elemento do site.
