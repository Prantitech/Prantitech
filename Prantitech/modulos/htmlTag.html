Tags para HTML

Tag Função
<!DOCTYPE><!DOCTYPE> Define o tipo de documento.
<a></a> A palavra marcada com essa tag terá um link para carregar outra página.
<br> Define uma quebra de linha.
<div></div> Não possui função estética, entretanto ajuda a demarcar e dividir bases no seu documento HTML.
<head> Contém a informação do documento, referências para sites externos, entre outros.
<header></header> Cria um cabeçalho.
<legend></legend> Define uma legenda para a tag <fieldset>
<img src=> Define uma imagem.
<input> Define um uma caixa para obter informações.
<meta> Define certas informações para um documento HTML.
<label></label> Define um comando para um <input>, auxiliando o usuario a prencher o espaço em branco de forma correta.
<textarea></textarea> Define um espaço em branco para digitar, porém é possivel controlar sua área, mudando seu tamanho.
<select></select> Define uma "lista" de alternativas.
<option></ option> Adiciona uma opção para uma lista de alternativas. É usado dentro da tag "<option>"
<strong></strong> Define um texto importante.
<ul></ul> Tudo dentro dessa tag sera informado como um elemento de uma lista não organizada.
<ol></ol> Tudo dentro dessa tag sera informado como um elemento de uma lista organizada.
<body></body> Toda a informação inserida na tag será exibida na tela.
<html></html> Tag estrutural para definir que todo o conteúdo dentro dela faz parte do HTML
<title></title> O texto inserido nessa tag irá ser posicionado no navegador.
<fieldset></fieldset> É usado para agrupar elementos, dentro de um formulário web.
